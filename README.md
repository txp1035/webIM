# webIM

## 分支说明

master：开发完成的分支，目前完成了 jQuery 版。

jQueryDev：jQuery 重构版，优化性能架构，增加新功能。（开发中）

reactDev 版：学习 react 用。（开发中）

vueDev 版：学习 vue 用。（开发中）
